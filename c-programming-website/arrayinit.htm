Array Initialization and Declaration

  • int array[6]={10,20,30,40,50,60};//Initialization

  • int array[6];//Declaration

  • scanf("%d ",&a[i]);//store data in array

  • printing the elements using index

  • printf("%d",array[0]);//prints the 10 Number

  • printf("%d",array[2]);//prints the 30 Number

  • char array[6]={'A','p','p','l','e','\0'};//Initialization

  • char array[6]//Declaration

  • printf("%c",array[0]);//prints A character

  • printf("%c",array[4]);//prints e character

  • \0 is null character

  • Ascii value of 0 is 48, 1 is 49 etc;

  • Ascii value of a is 97(lowercase alphabet) and Ascii value of z is 122(lowercase);

  • Ascii value of A is 65(upper case Alphabet), Ascii value of B is 66(uppercase);
